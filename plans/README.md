# Plans Directory

This directory contains all strategic and implementation plans for the Regulie project.

## Plans

- **MOBILE_QUICK_ENTRY_PLAN.md** - Plan for implementing mobile-first quick entry mode with voice-to-text and camera capture
- **regulie_rebranding_and_mongodb_encryption_plan.md** - Plan for rebranding from TherapyNotes to Regulie and implementing MongoDB encryption
- **therapy_notes_strategic_plan.md** - Original strategic plan for the therapy notes application (historical reference)

## Note

Some plans may reference the old "TherapyNotes" name. These are kept for historical reference and context.
